Find & Replace

How it works: parses each font's ToUnicode CMap, decodes CID-encoded text in Tj/TJ operators, replaces text in Unicode, re-encodes to CIDs. Only text bytes change — fonts, positions, colors, and everything else is untouched.

Limitation: Subset fonts only contain glyphs used in the original PDF. Replacement text must use characters that already appear in the document.
